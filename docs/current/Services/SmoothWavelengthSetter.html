/SmoothWavelengthSetter/CalculateMaxPossibleSpeedForSmoothMovement

Calculates max possible speed for scan with passed parameters, from start wavelength to end wavelength.

HTTP Method: PUT

The following is an example request Json body: 

{
	"CollectMotorPositionsDebugInfo":true,
	"CustomWavelengthStepSize":1.26743233E+15,
	"EndWavelength":1.26743233E+15,
	"IgnoreSeparationConfiguration":true,
	"Interaction":"String content",
	"MaxMotorPositionError":1.26743233E+15,
	"MaxSpeedError":1.26743233E+15,
	"StartWavelength":1.26743233E+15
}

The following is an example response Json body: 

{
	"IsSuccess":true,
	"LimitingFactorErrorMessage":"String content",
	"MaxSpeed":1.26743233E+15
}

/SmoothWavelengthSetter/CalculateMaxPossibleSpeedForSmoothMovementRoundTrip

Calculates max possible speed for scan with passed parameters, from start wavelength to end wavelength and back again.

The following is an example request Json body: 

{

	"CustomWavelengthStepSize":1.26743233E+15,
	"EndWavelength":1.26743233E+15,
	"IgnoreSeparationConfiguration":true,
	"Interaction":"String content",
	"MaxMotorPositionError":1.26743233E+15,
	"MaxSpeedError":1.26743233E+15,
	"StartWavelength":1.26743233E+15
}

The following is an example response Json body: 

{
	"IsSuccess":true,
	"LimitingFactorErrorMessage":"String content",
	"MaxSpeed":1.26743233E+15
}

/SmoothWavelengthSetter/Status

Get status of smooth wavelength setter.

HTTP Method: GET

The following is an example response Json body: 

{
	"IsRunning":true,
	"IsStopScheduled":true,
	"NanometersPerSecondIncludingStartStop":1.26743233E+15,
	"RepeatsLeft":2147483647
}

/SmoothWavelengthSetter/StopScanImmediately

Stops currently running scan immediately.

HTTP Method: PUT

/SmoothWavelengthSetter/StopScanOnEndpoint

Stops currently running scan on either end of range.

HTTP Method: PUT

/SmoothWavelengthSetter/StopScanOnStartingPosition

Stops currently running scan on starting position.

HTTP Method: PUT

Operations at /Motors

Uri Method Description
/ActualPosition GET Gets actual position of motor in steps. Postion is synced with hardware asynchronously.
/ActualPositionInUnits GET Gets actual position of motor in units (typically mm or deg). Postion is synced with hardware asynchronously.
/AllProperties GET Gets all properties of all motors.
/PropertiesThatChangeOften GET Gets often-changing properties properites of all motors.
/Stop POST Stop motor.
/StopAll POST Stop all motors.
/TargetPosition GET Gets target position of motor in steps.
PUT Sets target position in steps and starts moving motor towards it. Invalid positions are silently ignored. See /ActualPosition to know when (and if) motor reaches target position.
/TargetPositionInSuperUnits PUT Sets target position in super units and starts moving motor towards it. Invalid positions are silently ignored. See ActualPositionInSuperUnits to know when (and if) motor reaches target position.
/TargetPositionInUnits PUT Sets target position in units and starts moving motor towards it. Invalid positions are silently ignored. See ActualPositionInUnits to know when (and if) motor reaches target position.
/TargetPositionRelative PUT Sets target position in steps relative to current target position and starts moving motor towards it. Invalid positions are silently ignored. See /ActualPosition to know when (and if) motor reaches target position.
?id={motorIndex} GET Gets all properites of specific motor. If requested motor is not found or is not available due hardware failure error code 404 is returned.

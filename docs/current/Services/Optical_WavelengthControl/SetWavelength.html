Reference for /Optical/WavelengthControl/SetWavelength

Sets new wavelength. Interaction string will be used to select which interaction to use. It might be simple interaction string (SH-SIG) or expandable (#-SIG). Use * to match use all interactions.

HTTP Method: PUT

The following is an example request Json body: 

{
	"DontTouchShutterOnInteractionChange":true,
	"IgnoreSeparation":true,
	"SetForBackground":true,
	"Wavelength":1.26743233E+15,
	"Interaction":"String content",
	"Priorities":[{
		"Priority":2147483647,
		"Type":"String content"
	}]
}

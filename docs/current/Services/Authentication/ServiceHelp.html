Operations at /Authentication

Uri Method Description
/AuthenticationStatus GET Get status of authentication currently in progress.
/CallerHasAccess GET Returns true if caller can access all services. If he does not, call /StartAuthenticationByInterlock and repeatedly press interlock to grant access.
/StartAuthenticationByInterlock POST Start authentication. User has to repeatedly press interlock button to authenticate device. PC name will be used to identify user with fallback to IP address.
/StartAuthenticationByInterlockExplicitIP POST Start authentication for specific IP address. User has to repeatedly press interlock button to authenticate device.
/StartAuthenticationByInterlockExplicitPCName POST Start authentication for specific PC name. User has to repeatedly press interlock button to authenticate device.
/UsersWithAccessRights GET Returns list of user with access rights.

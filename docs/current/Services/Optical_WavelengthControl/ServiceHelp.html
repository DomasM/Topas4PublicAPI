Operations at /Optical/WavelengthControl

Uri Method Description
/ExpandedInteractions GET Gets all interactions with their calibration ranges in current optical configuration expanded to simple representation.
/FinishWavelengthSettingAfterUserActions PUT Client application should call this method after wavelength has been set and user's manual operations are completed.This method needs to be called onyl if setting wavelength required actions by user.If RestoreShutterState is true and shutter was open before setting wavelength it will be opened again.
/OpticalSystemData GET Gets Information about current and last set wavelength and some additional info.
/Output GET Gets current output information (wavelength, interaction, calculated motor positions and so on).
/Output/Interaction GET Gets current output interaction string. If wavelength setting is in progress returns target wavelengths interaction.
/Output/Wavelength GET Gets current output wavelength. If wavelength setting is in progress returns target wavelength.
/PossibleOutputsForWavelength POST Gets all possible ways to set specific wavelength.
/SetWavelength PUT Sets new wavelength. Interaction string will be used to select which interaction to use. It might be simple interaction string (SH-SIG) or expandable (#-SIG). Use * to match use all interactions.
/SetWavelengthUsingAnyInteraction PUT Sets new wavelength. Any of the possible interactions that support this wavelength will be used. If you need extra options use method SetWavelength.

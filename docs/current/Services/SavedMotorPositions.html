/Positions

Http method: POST

Add new postion to saved positions list.

Request JSON body example: 
{
  "Comment": "",
  "Folder": "",
  "GUID": "32056ae0-6fe7-4b07-8e82-48feae707f18",
  "MotorPositions": [
    {
      "Key": 19,
      "Value": 69
    },
    {
      "Key": 200,
      "Value": 157
    }
  ],
  "Name": "",
  "TimeCreated": "\/Date(1500038173415+0300)\/"
}

Response body is empty.

/SaveCurrent

Http method: POST

Add new postion to saved positions list. Returns GUID of it.

Request JSON body example: 
{
  "Folder": "",
  "Name": ""
}

Response JSON body example: 
"1d185c75"

/MoveMotorsToPosition

Http method: PUT

Move motors to positions as saved.

Request JSON body example: 
"759e0268"

Response body is empty.

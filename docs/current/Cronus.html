Introduction

Cronus control application is an end-user-oriented application for controlling OPO-based laser sources produced by Light Conversion. Interactive laser system control via the Cronus GUI is described in the User Manual of a particular system, this document covers local and remote control via the Cronus API.

Cronus provides a GUI for interactive laser system control and an API service for remote control, automation and advanced applications. Cronus operates on top of Topas4, whose server component carries out the actual hardware control. Depending on system configuration, one or more Topas4 server instances are used for hardware control and all must be running for the laser system to function properly. Running the Cronus application should start all the necessary server instances. From the API user standpoint, communication occurs between the client application and the Cronus app, all the necessary backend operations are handled by Cronus.

API Control Basics

API control is provided via a REST interface and COM port. COM port control is considered legacy method and is described here API calls between the client and the Cronus application are sessionless, i.e., there are no handles, control locks or opening/closing of devices. Multiple clients can use the REST service at the same time, and the Cronus GUI can be used at the same time as well.

The remote application sends API commands as HTTP REST requests to the Cronus application. A response is sent back to the remote application as part of an HTTP exchange. The contents of the REST messages are serialized using JSON.

Since REST commands are sessionless they are non-blocking. As a result, commands that take some time to finish (e.g. set wavelength, set GDD, open shutter) are only initiated by the REST command. You can track the progress of these actions by repeatedly polling their status.

Detailed Description of API Commands

  • General device control

Set Power Mode, etc

  • First output channel

Control the first output channel tunable in 680-960 nm range

  • Second output channel

Control the second output channel tunable in 960-1300 nm range. Commands are identical as for the first channel.

  • Third output channel

Control the third output channel fixed at 1025 nm

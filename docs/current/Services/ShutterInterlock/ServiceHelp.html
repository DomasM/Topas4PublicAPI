Operations at /ShutterInterlock

Uri Method Description
/CloseLaserOutputBeforeChangingShutter PUT If set to true will try to close laser output before opening/closing shutter. Output will be enabled after shutter state change. Laser control has to be configured (see /OutputControlWithLaserState).
/CloseShutter PUT Closes shutter.
/IsInterlockOpen GET Returns true if interlock is open and false otherwise.
/IsShutterOpen GET Returns true if shutter is open and false otherwise.
/OpenCloseShutter PUT Opens shutter if parameter is True, closes if false. If shutter can't be opened due safety reasons method returns HTTP code 200 (OK) and ignores command.
/OpenShutter PUT Opens shutter. If shutter can't be opened due safety reasons method returns HTTP code 200 (OK) and ignores command.
/OutputControlWithLaserState GET Get configuration and state of shutter control by lasers output control (i.e. pulse picker).
/State GET Get state of shutter & interlock unit.

Reference for /SmoothWavelengthSetter/CalculateMaxPossibleSpeedForSmoothMovementRoundTrip

Calculates max possible speed for scan with passed parameters, from start wavelength to end wavelength and back again.

HTTP Method: PUT

The following is an example request Json body: 

{
	"CollectMotorPositionsDebugInfo":true,
	"CustomWavelengthStepSize":1.26743233E+15,
	"EndWavelength":1.26743233E+15,
	"IgnoreSeparationConfiguration":true,
	"Interaction":"String content",
	"MaxMotorPositionError":1.26743233E+15,
	"MaxSpeedError":1.26743233E+15,
	"StartWavelength":1.26743233E+15
}

The following is an example response Json body: 

{
	"IsSuccess":true,
	"LimitingFactorErrorMessage":"String content",
	"MaxSpeed":1.26743233E+15
}
